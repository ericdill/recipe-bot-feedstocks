# recipe-bot-feedstocks

Test feedstock recipes generated by recipe-bot. Each subdirectory contains a
conda-build `meta.yaml` recipe that can be submitted to PBP for building.

## Structure

```
<package-name>/
  meta.yaml
```

## Usage with PBP

Each recipe directory can be referenced in a PBP build graph:

```yaml
feedstock_url: https://github.com/ericdill/recipe-bot-feedstocks
feedstock_ref: <commit-sha>
recipe_dir: <package-name>
```
